# Mark Shot

[English README](README.md)

<video src="https://github.com/jswysnemc/mark-shot/releases/download/v0.1.5/mark-shot.mp4" width="100%" controls></video>

`mark-shot` 是一款基于 Qt 6 开发的高性能、极速 Wayland 截图标注工具，专门针对 `niri` 等现代 Wayland 窗口管理器进行了原生级深度优化。

它通过 `grim` 瞬间截取无损屏幕画面，并打开自适应全屏标注覆盖层，为用户提供极其流畅的框选、标注、复制、保存以及高级贴图服务。

---

## 核心特色与技术创新

### 全面且高质量的标注工具箱
- **画笔与荧光笔**：支持平滑的自由线条绘制与自适应半透明高亮叠色。
- **几何矢量工具**：高精度的直线、矩形与椭圆绘制。
- **流线型尖锐箭头**：经过严密数学建模重构的六顶点经典箭头路径，尾部自然收缩呈尖细流线状，头部夹角被收窄至极佳的锐度，且支持防锯齿高质量渲染。
- **双重联动高级文本**：
  * **千级字号解锁**：字号上限完美放宽至 `1000px`，可通过滚轮或面板滑块极速调整。
  * **防意外折行保护**：引入了 `5%` 物理宽度安全缩放因子，克服了字体在不同字号下 Hinting 和 Kerning 渲染非线性的问题，在所有字号下都绝不产生意外自动换行。
  * **双重拖拽交互**：拖动文本框的**四个对角控制点**可实现字号与文本框的等比例联动无缝缩放；拖动文本框的**四条边控制线**则仅调节排版边界宽度以重新排版换行，而字号保持不变。
- **激光演示笔**：为教学与会议演示设计，绘制出的笔迹会随时间平滑渐变地自动融解消失，动态效果十足。
- **自增步骤序号**：轻点鼠标即可依次在屏幕上放置自增的步骤序号贴纸。
- **局部马赛克**：支持对敏感信息执行高质感毛玻璃区域虚化模糊。

### 悬浮贴图与交互优化 (Snipaste 级贴图体验)
- **局部一键贴图**：选中截图区域后，可将其作为一个独立、无边框且置顶的悬浮贴图窗口（`PinnedImageWindow`）固定于屏幕之上。
- **强大的贴图控制**：
  * **随心拖动**：鼠标左键按住贴图区域可随意在桌面移动。
  * **无级缩放**：滚动鼠标滚轮可对贴图进行无级缩放，或者双击 Ctrl 键一键重置大小。
  * **极速关闭**：双击鼠标左键或按下 `Esc` 键即可关闭贴图。
  * **右键菜单联动**：支持多角度旋转、另存为、一键重新复制到系统剪贴板、以及无级调整不透明度（0.2 - 1.0）。
- **右键误触精准拦截**：在 Qt 6 环境下，由于原生的上下文菜单点击机制，右键可能在弹出菜单时误触发菜单项。我们为此设计并安装了局部的 `LeftClickMenuFilter` 事件过滤器，精准拦截非左键在菜单范围内的动作，完美杜绝了误触，提供了极高的工业级人机交互品质。

### 原生 Wayland 与系统深度集成
- **Wayland 覆盖层渲染**：默认借助 `layer-shell-qt` 创建高速、无边框的全屏覆盖层；亦支持通过 `--xdg-window` 平滑降级至普通的 XDG 全屏窗口。
- **双桌面快捷方式与专属 SVG 图标**：
  * `mark-shot.desktop`：配置为系统全局截图工具，支持系统快捷键直接调用。
  * `mark-shot-edit.desktop`：注册为独立的“Mark Shot 图像编辑器”，可无缝集成到 Linux 文件管理器（如 Dolphin、Nautilus）的右键“使用其他程序打开”菜单中，用于直接打开并标注已有本地图像。
  * 装载了极高分辨率的 `mark-shot.svg` 与 `mark-shot-edit.svg` 系统矢量图标。

## 命令行接口 (CLI)

`mark-shot` 提供了极其灵活的命令行参数，用于满足不同的使用场景：

| 参数选项 | 功能描述 |
| :--- | :--- |
| `[file]` | **位置参数**：打开一个已有的本地图片文件进入标注模式，而不是捕获当前屏幕。一次仅允许打开一个文件。 |
| `-h`, `--help` | 显示帮助信息并退出。 |
| `-v`, `--version` | 显示当前版本信息并退出。 |
| `--all-outputs` | 捕获虚拟显示桌面的所有输出屏幕，而不是仅捕获当前的 Qt 活动屏幕。 |
| `--xdg-window` | 强制使用标准的 XDG 全屏普通窗口（xdg-shell）替代默认的 Wayland 覆盖层（layer-shell）。 |
| `--fullscreen` | 跳过选区步骤，直接对捕获的完整屏幕截图进行标注。 |

## 编译与安装

### 系统依赖 (以 Arch Linux 为例)

在构建前，请先安装以下必要依赖包：

```bash
sudo pacman -S --needed base-devel cmake ninja qt6-base qt6-wayland layer-shell-qt grim wl-clipboard
```

### 构建与编译

使用 CMake 和 Ninja 进行超快速的工程构建：

```bash
# 配置项目并输出到 build 目录
cmake -S . -B build -G Ninja -DCMAKE_BUILD_TYPE=Release

# 执行编译
cmake --build build
```

编译成功后，可执行文件将生成在 `./build/mark-shot` 路径下，可以直接运行测试。

### 安装与集成

要将软件安装到本地用户目录下并自动关联快捷方式和系统右键菜单，请运行：

```bash
cmake --install build --prefix "$HOME/.local"
```

安装完成后，你将在应用启动器和文件管理器的右键“打开方式”中看到 **Mark Shot Image Editor**。

## 交互快捷键与操作秘籍

在全屏标注模式下，您可以通过以下快捷键实现极速操作，极大地提升效率：

### 工具切换快捷键

| 快捷键 | 切换的目标工具 | 对应功能说明 |
| :---: | :--- | :--- |
| **V** | 移动 / 导航 (Move / Pan) | 在已有图像模式下，用于平移和拖动图像画布，双击 Ctrl 键复位。 |
| **S** | 选择 (Select) | 选中并移动、缩放或删除已绘制的矢量标注，或右键点击选区进行贴图。 |
| **P** | 画笔 (Pen) | 流畅的自由曲线绘制。 |
| **L** | 直线 (Line) | 绘制笔直的矢量线条。 |
| **H** | 荧光笔 (Highlighter) | 半透明的高亮覆盖，适合标记重点。 |
| **R** | 矩形 (Rectangle) | 绘制矩形线框。 |
| **E** | 椭圆 (Ellipse) | 绘制椭圆形线框。 |
| **A** | 箭头 (Arrow) | 绘制经典的六顶点尖细长锐角箭头。 |
| **T** | 文本 (Text) | 输入并编排富文本，支持 1000px 字号及拖拽联动。 |
| **N** | 序号 (Number) | 自动递增步骤序号标贴。 |
| **M** | 马赛克 (Mosaic) | 进行毛玻璃敏感区域虚化。 |
| **G** | 激光笔 (Laser) | 教学或展示使用的临时痕迹，会自动平滑消融。 |

### 全局操作快捷键

| 快捷键 | 触发动作 |
| :---: | :--- |
| **Esc** | 立即退出并关闭标注窗口。 |
| **Ctrl + C** | 确认所有文字编辑，并将当前截图/已标注选区复制到 Wayland 系统剪贴板。 |
| **Ctrl + S** 或 **Enter / Return** | 确认所有文字编辑，并保存当前截图。 |
| **Ctrl + Z** | 撤销上一步标注操作。 |
| **Ctrl + Y** 或 **Ctrl + Shift + Z** | 重做已被撤销的标注操作。 |
| **Backspace** 或 **Delete** | 在 **选择 (Select)** 工具激活且选中了某标注时，删除被选中的标注。 |
| **F** | 切换当前截图覆盖范围。 |

### 贴图窗口专属交互 (`PinnedImageWindow`)

| 交互手势 / 快捷键 | 动作效果 |
| :--- | :--- |
| **鼠标左键按住并拖动** | 自由平移和放置桌面贴图位置。 |
| **鼠标滚轮向上/向下** | 贴图窗口等比例无级放大/缩小。 |
| **双击鼠标左键** | 极速关闭该贴图窗口。 |
| **双击 Ctrl 键** | 一键复位贴图窗口的物理尺寸为原始比例。 |
| **鼠标右键单击** | 弹出功能菜单（包括旋转、调整透明度、保存、复制、关闭等）。 |
| **Esc 键** | 关闭当前获得焦点的贴图窗口。 |
